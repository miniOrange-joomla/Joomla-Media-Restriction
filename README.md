# Joomla Media Restriction Plugin – Secure File & Folder Access Control
Joomla Site Management: Media Restriction Extension

Secure your Joomla media library with the **miniOrange Joomla Media Restriction** plugin, designed to protect files and folders against unauthorized access. This plugin provides flexible restriction options based on **user roles**, **membership levels**, and **login status**, ensuring that sensitive media files like **eBooks**, **PDFs**, images, and more remain fully secure.

## Key Highlights:
- **File Type Support**  
  Restrict access to specific file extensions such as **PNG**, **JPG**, **GIF**, **PDF**, and more (unlimited extensions available in the licensed version).

- **Folder Restrictions**  
  Secure custom folders, subfolders, or Joomla’s default image folder — accessible only to logged-in users.

- **Advanced Redirection Options**  
  Redirect unauthorized users to Joomla **frontend/backend login pages** or an **SSO URL** for authentication.

---

## Features:

### **Free Version**
- Restrict access to limited file types (PNG, JPG, GIF, PDFs, etc.).
- Protect image folders from non-logged-in users.
- Compatible with **Apache** servers.

### **Licensed Version**
- Includes all Free features.
- Restrict unlimited file types and custom folder paths.
- Compatible with **Apache** and **NGINX** servers.
- **Role-Based Folder Restrictions**: Customize access based on user roles/groups.
- Integrates with **SAML** and **OAuth** plugins for authentication-based access control.
- Redirection to specific **login pages** or **SSO URLs** on access attempt.

---

## Benefits:
- Prevent unauthorized access to Joomla media files.
- Improve user experience with controlled access per user group or membership.
- Highly customizable to suit organizational security requirements.

---

## Quick Setup:
1. Install the plugin via **Extensions → Manage → Install**.
2. Configure folder/file access restrictions and redirection options.
3. Secure your media files based on **user groups**, **roles**, or **login status**.
